# Developer Portfolio

A modern, responsive developer portfolio built with [Next.js](https://nextjs.org/) and Tailwind CSS.

## Getting Started

First, install the dependencies. If you are on Windows, we recommend using `pnpm` to avoid filesystem lock issues:

```bash
npx pnpm install
# or
npm install
# or
yarn install
```

Then, run the development server:

```bash
npm run dev
# or
npx pnpm dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack
- **Framework**: Next.js (App Router)
- **Library**: React 19
- **Styling**: Tailwind CSS v4
- **Icons**: Lucide React
- **Language**: TypeScript
